O que é uma FlexBox?
Flexbox é uma tecnologia de layout em CSS que foi introduzida em 2009, como parte do CSS3. É um modelo de layout bidimensional, que permite organizar e posicionar elementos em um único eixo (linha ou coluna) ou em dois eixos (linhas e colunas).
